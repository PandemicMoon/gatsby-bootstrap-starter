## Getting Started
*Requires [Node.js](https://nodejs.org/en/).*

```bash
# install gatsby cli
npm install -g gatsby
# download files
git clone https://github.com/kmddigital/gatsby-bootstrap-starter.git
cd gatsby-bootstrap-starter
# start development server
gatsby develop
# build static site
gatsby build
# start http server
gatsby server
```
